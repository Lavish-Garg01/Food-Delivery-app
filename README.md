# FoodRush - Food Delivery Web Application

FoodRush is a full-stack food delivery web application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application allows users to browse food items, manage orders, and experience a responsive and user-friendly interface.

## Features

- User Authentication using JWT
- Secure Login and Registration
- Protected Routes
- Restaurant and Menu Management
- Order Management System
- Responsive User Interface
- MongoDB Atlas Integration
- Frontend Deployment on Vercel
- Backend Deployment on Render

## Installation

### Clone Repository

```bash
git clone https://github.com/Lavish-Garg01/Food-Delivery-app.git
```

### Install Frontend Dependencies

```bash
cd client
npm install
npm start
```

### Install Backend Dependencies

```bash
cd server
npm install
npm run dev
```
